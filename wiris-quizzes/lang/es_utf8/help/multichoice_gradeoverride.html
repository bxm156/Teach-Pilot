Esta es una opción de WIRIS Quizzes para preguntas mejoradas de matemáticas y ciencia.

Este campo es opcional. Si es utilizado, su valor debe ser el nombre de una variable de tipo booleano o numérico entre 0 y 1. El valor de esta variable será la calificación asociada a esta elección.

Visite manual de WIRIS Quizzes para más información.
