This is a WIRIS quizzes feature for enhanced math and science questions.

If activated, in the answer form appears a WIRIS CAS applet where students can perform auxiliar computations. In the case of Essay type questions, you may indicate that the applet replaces the HTML editor, so that the student must respond in the same WIRIS CAS. In Short answer type questions and if the option Formula Editor is choosen, will appear a button to copy the result of calculations into the formula editor.

Please go to WIRIS quizzes user manual for more information.
