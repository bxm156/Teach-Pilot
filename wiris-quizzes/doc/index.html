1        WIRIS Quizzes installation

1.1      Introduction

WIRIS Quizzes is a plugin for moodle aimed to improve the question types that involves mathematics and sciences in general.

In particular, it has the following features:

  1. Be able to add easily formulas in the questions and in the feedback.
  2. Generate questions enhanced by a program or algorithm:
    1. The formulas in the question or feedback can be generated by the program. They can be even random: thousands of different questions can be generated from a single one.
    2. The correct answer is also decided by the program.
  3. The short-answer questions are improved with the additional features:
    1. The user can type as an answer any formula using the WIRIS Formula Editor.
    2. The WIRIS Formula Editor can check in real time the syntax of the formula: missing parenthesis, incorrect us of the symbol *, etc.
    3. The answer can be checked smartly using any equivalence defined by the teacher (author).

1.2      Prerequisites

  • Moodle 1.9 or higher installed.
  • Plugin WIRIS for moodle 2.1.25 or higher installed.
  • WIRIS Quizzes package.

1.3      Starting

Unzip WIRIS Quizzes package on your moodle’s root directory. Now, you have two ways:

  • Install WIRIS Quizzes using the automatic installer (recommended).
  • Install WIRIS Quizzes manually (not recommended).
