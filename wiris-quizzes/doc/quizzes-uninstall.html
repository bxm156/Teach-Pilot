1        WIRIS Quizzes uninstallation

1.1      Advertisement

All quizzes and questions that uses WIRIS Quizzes module will stop working. You should remove or hide all questions and quizzes that uses WIRIS Quizzes before uninstall it.

1.2      Restoring moodle files

WIRIS Quizzes installation has modified some internal moodle files, but they were backuped. You should restore these files following these steps:

1.2.1      questionlib.php

  • Rename lib/questionlib.php to lib/questionlib.php.new
  • Rename lib/questionlib.php.old to lib/questionlib.php

1.2.2      question.php

  • Rename question/question.php to question/question.php.new
  • Rename question/question.php.old to question/question.php

1.3      Deleting directories

1.3.1      WIRIS Quizzes directory

Delete wiris-quizzes directory and its contents from your moodle root directory.

1.3.2      Quizzes installed

Go to question/type directory and delete these directories and its contents:

  • essaywiris
  • matchwiris
  • multichoicewiris
  • shortanswerwiris
  • truefalsewiris
