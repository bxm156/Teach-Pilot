--------------
To Install it:
--------------
    
    1- Copy the "moodle_google" folder to your \moodle\blocks directory    

    2- Cut/Copy all the files and folder from \moodle_google\lang folder into your Moodle's 
       lang directory - \moodle\lang... Overwrite it.

    3- After administratro login 
       Click "Notifications" on the home page of moodle and block will be installed.
    
  
--------------
To Use it:
--------------
  
    1- Go to your course page.

    2- Enable "turn editing on".

    3- Add 'moodle google' from the "Add block" menu.

    4- Start Using it.

    5- Use inbuilt help click the help logo (question mark).


--------------
Configuration:
--------------


I have used two kind of configuration for this block. One is Global Configuration that 
is for Administrator and Instance Configuration that is for the User.


    1- Global Configuration:-         (1) Provides the input for Google Search Api Key. 
      (Under Site Administration)     
				      (2) Provides the input for who can use this block. 
					  Authenticated User / Guest ?.
				      
				      (3) Hide the Moogle Logo in the block

    --------------------------------------------------------------------------------------------------


    2- Instance Configuration:-       Provides the various customizable parameters to pass to 
      (Press the Edit Button)  	      Google Search. Like number of results, 
				      Google Services etc. Check it out.
