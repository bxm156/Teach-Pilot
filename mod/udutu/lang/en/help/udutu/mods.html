myUdutu

    myUdutu is a free online course authoring environment that is designed to enable subject matter experts and instructional designers to rapidly build courses and collaborate online on the course creation process

    The myUdutu module allows you to import your courses from a myUdutu server easily and use them like Moodle SCORM activities. More information about creating myUdutu courses can be found on the myUdutu web site.

    If you don't have a myUdutu account yet, you can create one for free at www.myudutu.com

    A forum for myUdutu can be found here. You must create an account on myUdutu to post in the forum.
