User Auto Registration

Auto registration allows avatars to have a new Moodle account automatically registered for them by Sloodle. This saves time, and prevents the need for SL users to interact directly with Moodle at all. It must first be enabled at the site-level by an administrator, and it must then be enabled for each course as well

If you enable auto-registration, then you will probably want to enable auto-enrolment as well.

Warning!

Auto-registration only works with normal Moodle accounts. It may not function correctly if you have setup Moodle networking or external user databases.
